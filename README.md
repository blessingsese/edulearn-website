# 🎓 EduLearn – Online Learning Platform

A fully responsive, multi-page educational web application built with HTML5, CSS3, JavaScript (ES6+), and Bootstrap 5.

## 🌐 Live Site
#  https://blessingsese.github.io/edulearn-website/

## 📄 Pages
- **Home** – Hero section, features, and call to action
- **About** – Mission, team, and stats
- **Courses** – Filterable course catalogue
- **Course Detail** – Full course info with accordion curriculum and enroll modal
- **Blog** – Articles with live search
- **Contact** – Form with JavaScript validation
- **FAQ** – Accordion Q&A sections
- **Testimonials** – Student reviews and ratings

## ⚙️ JavaScript Features
1. Contact form validation (name, email, subject, message)
2. Dark / Light mode toggle (persists across pages)
3. Course category filter
4. Blog live search

## 🛠️ Technologies Used
- HTML5 (semantic elements)
- CSS3 (Flexbox, custom properties, media queries)
- JavaScript ES6+
- Bootstrap 5.3
