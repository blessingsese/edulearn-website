Course Curriculum

Introduction to the web, HTML tags, forms, tables, semantic elements (header, nav, section, footer). Project: Build a personal profile page.

Bootstrap grid, components, and utilities. Git version control and deploying to GitHub Pages. Final project: Multi-page website.
